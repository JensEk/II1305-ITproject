About Tomatori

Tomatori is a discord-bot based application which works as a timer. This bot can be invited to any server on Discord and with commands you can start timers for your studies. This bot has the ability to mute you on that specific server and tracks the amount of time you have disciplined yourself to only study

You will receive Tomato Points for the minutes that you use the Tomatori bot. These points will be used in a web-based game where you can spend it as lives for jumping or for epic skin purchases at the game shop
